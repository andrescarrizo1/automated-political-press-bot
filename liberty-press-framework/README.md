# Liberty Press Framework

Sistema de automatización de prensa política para 
municipios y medios de comunicación de Argentina.

## Estructura del proyecto
- _template/ → base para nuevos clientes
- clientes/ → configuración por municipio y perfil político
- docs/ → propuestas comerciales y onboarding

## Para agregar un cliente nuevo
1. Copiar _template/ a clientes/[nombre_municipio]/
2. Completar config.json con datos del cliente
3. Agregar fuentes en fuentes/
4. Usar checklist en docs/onboarding/
